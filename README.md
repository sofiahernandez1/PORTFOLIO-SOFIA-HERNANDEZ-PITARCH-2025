# PORTFOLIO SOFIA HERNANDEZ PITARCH 2025

Este es el proyecto final de Sofía Hernández Pitarch en la asignatura de Maquetación Web con hojas de estilo, el cual consiste en la creación de un portfolio personal. En él, se encuentra una breve descripción de ella como diseñadora y algunos de sus mejores proyectos con informacion e imagenes de cada uno de ellos.

## Stack tecnológico

- HTML5
- CSS3 (arquitectura por capas: base/layout/components)
- JavaScript (Vanilla)
- Bootstrap 5 (CDN)

## Instrucciones de configuración para desarrollo local

1. Clona el repositorio desde GitHub: https://github.com/sofiahernandez1/PORTFOLIO-SOFIA-HERNANDEZ-PITARCH-2025.git
2. Accede a la carpeta del proyecto:
   cd PORTFOLIO-SOFIA-HERNANDEZ-PITARCH-2025
3. Abre el proyecto en Visual Studio Code:
   code .
4. Abre el archivo index.html utilizando la extensión Live Server.

## Guía de personalización

- Los colores principales del sitio se pueden modificar desde las variables CSS definidas en los archivos de estilos.
- Las tipografías pueden cambiarse editando las fuentes importadas en la carpeta `assets/fonts` o mediante Google Fonts.
- Las imágenes y vídeos se gestionan desde la carpeta `assets/images`.
- Los textos y contenidos de cada sección se pueden editar directamente en el archivo `index.html`.

## Política de uso de agentes de IA

Durante el desarrollo de este portfolio se ha utilizado un asistente de Inteligencia Artificial como herramienta de apoyo puntual, no como generador completo del proyecto.

La IA se empleó principalmente en tareas de **planificación, organización y revisión**, como la estructuración del proyecto, la aclaración de conceptos técnicos (HTML, CSS y JavaScript), la mejora de la organización del código y la verificación del cumplimiento de los requisitos de entrega.

En ningún caso la Inteligencia Artificial se utilizó para generar de forma automática la totalidad del código del sitio web. La implementación final, las decisiones de diseño y el desarrollo del proyecto fueron realizados de manera manual y consciente, comprendiendo y adaptando cada parte del código.

El uso de la IA se ha realizado siguiendo un flujo en dos fases (planificación e implementación), y las interacciones relevantes se han documentado conforme a las indicaciones de la asignatura.
