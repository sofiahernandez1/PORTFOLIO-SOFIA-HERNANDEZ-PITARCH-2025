# Navegación responsive y estructura del header

## Fase 1: Planificación (uso de IA)

### Prompt realizado a la IA

> Estoy haciendo un portffolio web y necesito que me ayudes a planificar la estructura para hacer una navegación responsive. En ella me gustaría incluir un menú hamburguesa para el formato móvil y una navbar fija durante toda la web, utilizando HTML, CSS y Bootstrap.

### Respuesta de la IA (plan)

La IA ha propuesto estructurar la navegación utilizando una etiqueta `<header>` con un `<nav>` semántico, apoyándose en Bootstrap para el comportamiento responsive del menú. Ha recomendado el uso de clases utilitarias para el layout, un botón hamburguesa para pantallas pequeñas y una barra fija para escritorio, así como separar la lógica visual (CSS) del comportamiento (JavaScript).

## Fase 2: Implementación

Se ha implementado una navegación responsive con menú hamburguesa en móvil y barra fija en escritorio. Se ha utilizado Bootstrap como apoyo para el comportamiento del menú y se ha personalizado el diseño mediante CSS, adaptándolo al diseño visual del portfolio.
