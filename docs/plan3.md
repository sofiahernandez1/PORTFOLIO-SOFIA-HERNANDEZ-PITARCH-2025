# Documentación y preparación para la entrega

## Fase 1: Planificación (uso de IA)

### Prompt realizado a la IA

> Necesito ayuda para revisar los requisitos de entrega de un portfolio web académico y asegurar que mi proyecto cumple con la documentación, licencia, estructura de carpetas y buenas prácticas exigidas.

### Respuesta de la IA (plan)

La IA ha indicado la necesidad de incluir un README completo con enlace al despliegue, descripción del proyecto, stack tecnológico, instrucciones de desarrollo local, guía de personalización, capturas y créditos. También ha recomendado añadir una licencia MIT y documentar el uso de IA mediante archivos de planificación en la carpeta `docs/`.

## Fase 2: Implementación

Se ha redactado el README siguiendo los requisitos establecidos, se ha añadido la licencia MIT en la raíz del proyecto y se ha documentado el uso de la IA mediante archivos de planificación, asegurando una entrega clara, ordenada y conforme al enunciado.
