# Organización de secciones y layout del portfolio

## Fase 1: Planificación (uso de IA)

### Prompt realizado a la IA

> Quiero organizar mi portfolio con varias secciones (hero, menu grande, about, poster, album, editorial, 3d, galería y contacto). ¿Cómo puedo estructurar el HTML y el layout para que sea claro, responsive y fácil de mantener?

### Respuesta de la IA (plan)

La IA ha sugerido dividir el contenido en secciones semánticas (`<section>`) con una jerarquía clara de encabezados. Ha recomendado utilizar Flexbox y Grid para los layouts, aplicar un enfoque mobile-first y mantener una separación clara entre estructura HTML, estilos CSS y comportamiento JavaScript.

## Fase 2: Implementación

Se ha estructurado el portfolio en secciones diferenciadas (hero, about, poster, álbum, editorial, galería y contacto), aplicando layouts flexibles y responsive. La organización final facilita la lectura del código y la adaptación a distintos tamaños de pantalla.
